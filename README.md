# 📘 Software Engineering Sessional

---

## 📌 Project Overview

This project is a **Full Stack E-commerce Web Application** developed for the **Software Engineering Sessional** course.

The system allows users to:

* Browse products
* Add items to cart
* Place orders
* Manage accounts

An admin panel is also included to manage products, users, and orders.
